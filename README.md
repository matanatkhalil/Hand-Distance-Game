# Hand-Distance-Game
A computer-vision mini game built with OpenCV and MediaPipe hand tracking.
The player interacts with on-screen targets by using hand distance from the camera.

## About this project
This project was created as a learning exercise by following a tutorial and then adapting the code.
Several parts were modified and debugged to improve usability across different screen sizes.

## Based on
This project is based on the following YouTube tutorial:
[Hand Distance Measurement with Normal Webcam + Game | OpenCV Python](https://www.youtube.com/watch?v=NGQgRH2_kq8)
